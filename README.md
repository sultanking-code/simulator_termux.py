# مشروع محاكاة اختبار اختراق الأجهزة على Termux

## نظرة عامة
هذا المشروع عبارة عن **محاكاة بيئة اختبار اختراق متقدمة** تعمل على أجهزة Android باستخدام **Termux**.  
يهدف المشروع إلى:  
- تعليم مفاهيم الأمن السيبراني بشكل عملي وآمن.  
- محاكاة الاختبارات الداخلية، الخارجية، واللاسلكية للأجهزة.  
- محاكاة إرسال “بايلود” بطريقة وهمية وآمنة.  
- توليد تقارير نهائية وجاهزة للتسليم بصيغة CSV.  

> جميع العمليات في المشروع هي **محاكاة آمنة بالكامل** ولا تؤثر على أي جهاز فعلي أو بيانات حقيقية.

---

## المتطلبات

- Termux على جهاز Android.  
- Python 3.9 أو أحدث.  
- مكتبة Python: `rich` (لواجهة النص التفاعلية).
